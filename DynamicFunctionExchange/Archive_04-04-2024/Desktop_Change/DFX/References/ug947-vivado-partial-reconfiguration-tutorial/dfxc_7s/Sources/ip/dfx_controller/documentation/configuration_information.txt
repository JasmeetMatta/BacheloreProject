ID Mapping
=============================
Virtual Socket Manager vs_shift is assigned ID 0
  Reconfigurable Module rm_shift_left is assigned ID 0
  Reconfigurable Module rm_shift_right is assigned ID 1

Virtual Socket Manager vs_count is assigned ID 1
  Reconfigurable Module rm_count_up is assigned ID 0
  Reconfigurable Module rm_count_down is assigned ID 1


Address Structure
=============================
The addresses are structured as follows:
  address[ 8: 8]  : Selects the Virtual Socket Manager
  address[ 7: 6]  : Selects the Register Bank
  address[ 5: 2]  : Selects the Register within the bank
